# Data Structure Planning Table

| Operation | Data Structure Used | Reason for Choosing |
|----------|--------------------|--------------------|
| Add Song to Playlist | Circular Linked List | Allows continuous music playback |
| Delete Song | Circular Linked List | Efficient node removal |
| Display Playlist | Circular Linked List | Traverse whole playlist easily |
| Play Next Song | Circular Linked List | Automatically moves to first song after last |
| Play Previous Song | Circular Linked List | Supports backward traversal through loop |
| Remove Duplicate Songs | Circular Linked List | Ensures unique playlist efficiently |
